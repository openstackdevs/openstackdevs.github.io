# Favicons

Add favicon files to this directory:

- favicon.ico (32x32)
- favicon-16x16.png (16x16)
- favicon-32x32.png (32x32)
- apple-touch-icon.png (180x180)
- safari-pinned-tab.svg (SVG)

Use these tools:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

For an OpenStack-themed logo, consider using cloud/server icons from:
- https://fontawesome.com/icons (cloud server icon)
- https://thenounproject.com/ (server rack icons)
