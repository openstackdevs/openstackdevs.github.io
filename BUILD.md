# Build verification

This site is built with Hugo and the PaperMod theme. The PaperMod theme is included as a git submodule.

## Local build

To build locally:
```bash
git submodule update --init --recursive
hugo --minify
```

## Theme

The site uses the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme.
